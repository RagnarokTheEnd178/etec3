Banco de Dados

É uma matéria que consiste no aprendizado e criação de bancos de dados, que são conjuntos de dados armazenados de forma coerente. Formar informações e ligar conceitos de forma compreensiva é essencial para o aprendizado nesta matéria. Os dados nos tipos mais comuns de bancos de dados em operação atualmente são modelados em linhas e colunas em uma série de tabelas para tornar o processamento e a consulta de dados eficientes. Os dados podem ser facilmente acessados, gerenciados, modificados, atualizados, controlados e organizados. A maioria dos bancos de dados usa a linguagem de consulta estruturada (SQL) para escrever e consultar dados.
